Security

Съществуват два начина за authentication при websocket-ите. Това са User-based и TLS-based моделите.

Защитата на WebSocket Endpoint-ите става по стандартния начин за защита на ресурси в описателният файл (web.xml) на уеб приложението.

Пример с BASIC authentication:

Незащитен Endpoint

var wsUri = "ws://" + document.location.host + document.location.pathname + "websocket";

Защитен Endpoint

var wsUri = "ws://userName:password@" + document.location.host + document.location.pathname + "websocket";

HTTPS, WSS и TLS

Незащитен Endpoint

var wsUri = "ws://" + document.location.host + document.location.pathname + "websocket";

Защитен Endpoint

var wsUri = "wss://" + document.location.host + document.location.pathname + "websocket";
